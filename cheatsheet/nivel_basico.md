## 🟢 Nível 1 — Navegação e Exploração Básica

> Primeiros comandos que todo iniciante deve aprender para abrir, visualizar e explorar um banco de dados.

| Comando | Função |
|---|---|
| `.help` | Lista todos os comandos disponíveis no SQLite. |
| `.open arquivo.sqlite` | Abre um banco de dados SQLite. |
| `.databases` | Mostra os bancos atualmente conectados. |
| `.tables` | Lista todas as tabelas e views do banco. |
| `.schema` | Exibe toda a estrutura do banco de dados. |
| `.schema TABELA` | Mostra apenas a estrutura da tabela especificada. |
| `.quit` | Encerra a sessão do SQLite. |
| `.exit` | Sai do SQLite (equivalente ao `.quit`). |
| `.version` | Mostra a versão instalada do SQLite. |
